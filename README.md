# AI-Powered Predictive Vehicle Maintenance System

An intelligent machine learning-based system designed to predict vehicle maintenance requirements and detect potential failures before they occur. This project focuses on predictive analytics, anomaly detection, and data-driven maintenance forecasting to improve vehicle reliability and reduce operational downtime.

---

## Features

- Predictive maintenance using Machine Learning
- Vehicle failure prediction and anomaly detection
- Data preprocessing and feature engineering
- Real-time maintenance forecasting
- Performance analysis and visualization
- Scalable and intelligent maintenance workflow

## Project Objective

Traditional vehicle maintenance methods often rely on fixed schedules or manual inspections, which may lead to unexpected failures and increased operational costs.

This project aims to:
- Predict maintenance needs proactively
- Identify abnormal vehicle behavior
- Reduce unexpected breakdowns
- Improve operational efficiency
- Enhance vehicle reliability using AI-driven analytics

---

##  System Workflow

1. Vehicle performance data collection
2. Data preprocessing and cleaning
3. Feature extraction and engineering
4. Machine learning model training
5. Failure prediction and anomaly detection
6. Maintenance recommendation generation

## Running the Project

### Clone the Repository

```bash
git clone https://github.com/PojashriJM/AI-Powered-Predictive-Vehicle-Maintenance-System.git
cd AI-Powered-Predictive-Vehicle-Maintenance-System
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Streamlit Application

```bash
streamlit run app.py
```
